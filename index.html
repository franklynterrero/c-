Conceptos de Programación

  • Programa: Un conjunto de instrucciones escritas en un lenguaje de programación que realiza una tarea específica o resuelve un problema. Los programas pueden ser simples (como mostrar un mensaje) o complejos (como gestionar un sistema).
  • Tipos de programas: Existen dos tipos principales: el software de sistema, que gestiona el hardware y el software, y el software de aplicación, que ayuda a los usuarios a realizar tareas específicas (como un editor de texto).
  • Código fuente: Es el conjunto de instrucciones escritas por un programador en un lenguaje de programación que luego son convertidas en código máquina o código ejecutable para su funcionamiento.
  • Código objeto: El código fuente compilado o traducido a un formato que la máquina puede entender y ejecutar. Es un paso intermedio antes de generar el código ejecutable.
  • Software de sistema: Son los programas que gestionan y controlan los recursos del hardware, como sistemas operativos o controladores, permitiendo que otros programas funcionen correctamente.
  • Software de aplicación: Programas diseñados para ayudar a los usuarios a realizar tareas específicas, como escribir documentos, navegar por Internet o jugar, ejemplo: Word, Excel o navegadores.
  • Palabras reservadas: Son palabras que tienen un significado especial en un lenguaje de programación y no pueden ser utilizadas para otros fines, como definir variables o funciones. Ejemplos: `int`, `for`, `if`.
  • Comentarios: Son anotaciones en el código fuente que no son ejecutadas por el programa, pero sirven para documentar y explicar partes del código. Ayudan a la comprensión del mismo por parte de otros programadores.
  • Concepto de librería: Es un conjunto de funciones o clases predefinidas que pueden ser reutilizadas en distintos programas. Las librerías facilitan el desarrollo al no tener que escribir código repetido.
  • Concepto de Método o Función: Un método o función es un bloque de código que realiza una tarea específica dentro de un programa. Permiten dividir el código en secciones más manejables y reutilizables.
  • Método principal (main): Es el punto de entrada de un programa en lenguajes como C# o Java. El método `main` es el primero en ejecutarse y generalmente controla el flujo de la aplicación.
  • Conceptos de Datos: Los datos son la información que procesa un programa, y pueden ser números, texto o estructuras más complejas como listas o objetos. Se representan mediante tipos de datos en un lenguaje de programación.
  • Variables: Son contenedores que almacenan valores que pueden cambiar durante la ejecución del programa. Se utilizan para guardar datos temporales o resultados intermedios.
  • Constantes: Son valores que no cambian durante la ejecución de un programa. Se definen al inicio y se mantienen iguales a lo largo de todo el código.
  • Tipos de datos: Definen el tipo de información que puede almacenar una variable, como números enteros, decimales, caracteres o booleanos. Cada lenguaje de programación tiene sus propios tipos de datos.
  • Numéricos: Entero: Son tipos de datos que almacenan números sin parte decimal, como 5, -100 o 2024. Los enteros se usan para representar cantidades que no requieren fracciones.
